purplecoat.js

Purplecoat.js lets you create labeled overlays that can be triggered with a click. And you don't need to write a single line of Javascript.

Download on GitHub →

bower install purplecoat.js

Tweet

Usage

Step 0: Add purplecoat.js along with jQuery.

<script type="text/javascript" src="https://code.jquery.com/jquery-2.1.1.min.js"></script>
<script type="text/javascript" src="purplecoat.js"></script>

Step 1: Add data-purplecoat="foo" and data-purplecoat-label="Hello!" to any element.

<img data-purplecoat="foo"
     data-purplecoat-label="Hello!"
     src="holder.js/200x150/#FE97CA:#D64F91">

Step 2: Add data-purplecoat-toggle="foo" to any clickable element.

Now, go ahead and click this button to show the overlay label. Click again to hide the label.

<button data-purplecoat-toggle="foo" class="btn btn-default">
Click Me!
</button>

(Explanation: The value of deta-purplecoat-toggle must match the value of data-purplecoat to make this all work.)

Customizations

You can create multiple triggers:

<button data-purplecoat-toggle="odd"
        class="btn btn-default btn-lg">Odd</button>
<button data-purplecoat-toggle="even"
        class="btn btn-default btn-lg">Even</button>

<img data-purplecoat="odd"
     data-purplecoat-label="1 is odd"
     src="holder.js/80x80/#FE97CA:#D64F91/text:1">
<img data-purplecoat="even"
     data-purplecoat-label="2 is even"
     src="holder.js/80x80/#FE97CA:#D64F91/text:2">
<img data-purplecoat="odd"
     data-purplecoat-label="3 is odd"
     src="holder.js/80x80/#FE97CA:#D64F91/text:3">

You can also customize the overlay color via data-purplecoat-color:

<button data-purplecoat-toggle="bluecoat"
        data-purplecoat-color="rgba(63, 159, 252, 0.8)"
        class="btn btn-default btn-lg">Click Me!</button>

<img data-purplecoat="bluecoat"
     data-purplecoat-label="Hello!"
     src="holder.js/200x150/#FE97CA:#D64F91">
